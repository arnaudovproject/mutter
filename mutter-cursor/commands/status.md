# mutter status

Run the **status** workflow from the Mutter plugin.

1. Open `.mutter/core/project.md` for orientation.
2. Invoke the matching skill **status** so full instructions load.
3. From the repo root, run `python3 scripts/mutter.py tasks-status` (and `tasks-status --task …` when scoped).

For slash-style usage in Claude Code, use `/mutter:status`. In Cursor, use the Command palette entry or ask the agent to follow the **status** skill.
