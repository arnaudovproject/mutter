# mutter help

Run the **help** workflow from the Mutter plugin.

1. Open `.mutter/core/project.md` for orientation.
2. Invoke the matching skill **help** (same name) so full instructions load.
3. Keep responses incremental; update `.mutter/metadata/scan-state.json`, tasks, and ADRs when your actions change project truth.

For slash-style usage in Claude Code, use `/mutter:help`. In Cursor, use the Command palette entry or ask the agent to follow the **help** skill.
