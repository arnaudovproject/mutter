# mutter prd

Run the **prd** workflow from the Mutter plugin.

1. Open `.mutter/core/project.md` for orientation.
2. Invoke the matching skill **prd** so full instructions load.
3. Prefer **`python3 scripts/mutter.py prd-init`** once per workspace and **`validate-prd`** after substantive edits.

For slash-style usage in Claude Code, use `/mutter:prd`. In Cursor, use the Command palette entry or ask the agent to follow the **prd** skill.
