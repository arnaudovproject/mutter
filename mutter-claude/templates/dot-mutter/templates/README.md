# Templates

Task templates, plan skeletons, ADR templates for copy into `tasks/current/` or `plans/`.

| File | Use |
|------|-----|
| `TASK.md` | Copy to `tasks/current/<slug>.md` when creating work; fill **Acceptance** and **Verify** before deep execution. |
| `PRD.md` | Scaffold via **`python3 scripts/mutter.py prd-init`** → `.mutter/prd/PRD.md`; refine with **`/mutter:prd`**. |
